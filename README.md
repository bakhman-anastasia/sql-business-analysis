# SQL-анализ бизнес-метрик

## Описание проекта
Проект посвящён анализу бизнес-данных с использованием SQL.
Цель — получить ключевые метрики, выявить изменения в показателях
и проанализировать структуру продаж / клиентов.

## Используемые инструменты
- SQL
- PostgreSQL

## Что было сделано
- Анализ данных из нескольких таблиц
- Оконные функцие, CTE
- Использование JOIN для объединения данных
- Расчёт агрегированных метрик (выручка, количество заказов, средний чек)
- Фильтрация и группировка данных по ключевым признакам

## Результаты
- Получены основные бизнес-метрики
- Выявлены различия между сегментами
- Сформированы выводы для дальнейшего анализа
